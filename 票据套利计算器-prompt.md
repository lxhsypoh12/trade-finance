# 角色
你是一名资深票据业务成本会计，精通银行承兑汇票套利交易的成本核算与利润分析。

# 业务场景
贸易公司利用"拆借资金 → 定期存款质押 → 开具银行承兑汇票 → 票据贴现 → 还款"进行套利。具体流程：
1. A贸易公司从外部拆借金额【X】万元，拆借日利率【r】%/天
2. 将拆借资金在B银行存入【X】万元、【Y】年期定期存款，年利率【i】
3. 以该存单质押，在B银行开具等额【X】万元的银行承兑汇票，支付开票手续费【F1】元
4. 将银票支付给上游C贸易公司作为货款
5. C公司将银票在D银行贴现，贴现年化利率【j】，支付贴现手续费【F2】元，获得现金
6. 贴现所得现金用于归还拆借资金，从开票到还款共历时【z】天
7. 公司还需分摊运营成本（含税），费率【a】%，按开票金额计算
8. 城投公司因指定存款银行，向我司支付补贴，费率【Z】%，按票面金额计算
9. 存款利息为到期一次性支付，城投公司补偿后置利息收入，费率【H】%，按存款利息×期限计算
10. A公司最终利润 = 存款利息 − 贴现利息 − 手续费 − 拆借成本 − 运营成本 + 补贴收入 + 后置利息

# 任务
创建一个可在浏览器中运行的 HTML 套利利润计算器。

# 输入变量（用户手动填写）
| 变量 | 含义 | 单位 |
|------|------|------|
| X | 票面金额（也是拆借本金） | 万元 |
| Y | 存款/票期 | 年 |
| i | 定期存款年利率 | % |
| j | 贴现年化利率 | % |
| F1 | 开票手续费（固定） | 元 |
| F2 | 贴现手续费（固定） | 元 |
| r | 拆借资金日利率 | %/天 |
| z | 开票到还款天数 | 天 |
| a | 公司运营成本率（含税） | % |
| Z | 城投补贴费率 | % |
| H | 后置利息率 | % |

# 计算逻辑
1. 存款利息 = X × 10000 × (i / 100) × Y
2. 贴现利息 = X × 10000 × (j / 100) × Y
3. 价差（毛利润）= 存款利息 − 贴现利息
4. 开票手续费 = F1
5. 贴现手续费 = F2
6. 拆借资金成本 = X × 10000 × (r / 100) × z
7. 公司运营成本 = X × 10000 × (a / 100)
8. 城投补贴收入 = X × 10000 × (Z / 100)
9. 后置利息收入 = 存款利息 × (H / 100) × Y
10. 净利润 = 价差 − F1 − F2 − 拆借成本 − 运营成本 + 补贴收入 + 后置利息

# 界面要求：双栏分步展开式
## 左栏（输入区）
- 11个输入框，纵向排列，每个变量配有中文标签和单位提示
- 变量分四组，组间用虚线分隔：
  - 基础参数（X / Y / i / j）
  - 银行费用（F1 / F2）
  - 资金成本（r / z / a）
  - 补贴收入（Z / H）
- 默认值：X=100, Y=1, i=3.5, j=2.8, F1=5000, F2=3000, r=0.02, z=30, a=0.5, Z=0.5, H=0.3
- 输入框支持实时修改，修改后自动重算
- 输入验证：X/Y/z 必须 > 0，其余变量 ≥ 0

## 右栏（计算明细区）
从上到下分步展开，每步一个卡片：

卡片1 ─ **📋 输入确认**：回显11个参数值，带单位
卡片2 ─ **💰 存款利息**：公式代入，显示结果
卡片3 ─ **📉 贴现成本**：贴现利息 + 价差（盈利/倒挂）
卡片4 ─ **🧾 银行手续费**：F1 + F2 合计
卡片5 ─ **🏦 拆借资金成本**：公式 `X×10000×(r÷100)×z`
卡片6 ─ **🏢 公司运营成本**：公式 `X×10000×(a÷100)`
卡片7 ─ **🏛️ 城投公司补贴收入**：公式 `X×10000×(Z÷100)`，绿色收入
卡片8 ─ **⏳ 后置利息收入**：公式 `存款利息×(H÷100)×Y`，绿色收入
卡片9 ─ **📊 净利润**：完整公式代入 + 正负颜色 + 年化收益率

# 技术要求
