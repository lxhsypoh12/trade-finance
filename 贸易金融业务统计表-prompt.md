# 贸易金融业务统计系统

## 1. 角色

你是一名业务统计系统架构师，负责为公司搭建一个**贸易金融业务全生命周期统计系统**，覆盖从投标、中标到服务完成的完整业务链路。

## 2. 系统总览

### 2.1 业务流程

```
投标阶段 ──中标──▶ 中标阶段 ──服务完成──▶ 完成阶段
   │                 │                    │
   └──未中标──▶ 删除   │                    │
                       │                    ▼
                       │            汇总公司总利润（自动计算）
```

### 2.2 三阶段统计表职责

| 阶段 | 核心职责 | 数据去向 |
|------|---------|---------|
| **投标阶段** | 记录业务基本要素，供业务同事寻找合作方并完成报价 | 中标→迁移至"中标阶段"；未中标→删除 |
| **中标阶段** | 跟踪中标业务的服务完成情况，记录贸易合作方及中标数据 | 服务完成→迁移至"完成阶段" |
| **完成阶段** | 记录实际收入/成本/利润，供业务领导查看板块整体利润 | 终点，数据沉淀 |

### 3.2 投标阶段业务统计表

**视图标签：** `投标阶段业务统计表`

**输入字段（7项）：**
- 区域 — 文本输入
- 城投公司 — 文本输入
- 金融机构 — 文本输入
- 业务规模 — 数字输入，单位：万元
- 业务类型 — 下拉选择（参考选项：银承-回报 / 银承-供应链 / 信用证-回报 / 信用证-供应链 / 跑量/受托支付 / 存款 / 时点）
- 时间 — 年/月 格式输入（如 2026/06）
- 负责人 — 文本输入

**表格操作：**
- 每行最左侧有 **复选框**，支持多选
- 表格上方有「添加记录」按钮，点击后在表格末尾新增一行空白编辑行

**底部按钮（2个）：**
- **「未中标」按钮** — 删除所有被勾选的记录（操作前弹出确认框："确认删除选中的 X 条记录？"）
- **「中标」按钮** — 将勾选的记录迁移至"中标阶段业务统计表"，同时从本表删除（操作前弹出确认框："确认将选中的 X 条记录迁移至中标阶段？"）

### 3.3 中标阶段业务统计表

**视图标签：** `中标阶段业务统计表`

**字段来源：**

| 字段 | 来源 |
|------|------|
| 区域、城投公司、金融机构、业务规模、业务类型、时间、负责人 | 从投标阶段迁移（只读，不可编辑） |
| 贸易方、贸易成本、中标费率、中标金额 | 手动输入，可编辑 |
| 预计公司利润 | 自动计算：`中标金额 − 贸易成本`，只读 |

**表格操作：**
- 每行最左侧有 **复选框**，支持多选
- 迁移过来的7个基础字段显示为只读样式（灰色背景）
- 手动输入的4个字段为可编辑输入框

**底部按钮（1个）：**
- **「服务完成」按钮** — 将勾选的记录迁移至"完成阶段业务统计表"，同时从本表删除（操作前弹出确认框）

### 3.4 完成阶段业务统计表

**视图标签：** `完成阶段业务统计表`

**字段来源：**

| 字段 | 来源 |
|------|------|
| 区域、城投公司、金融机构、业务规模、业务类型、时间、负责人 | 从中标阶段迁移（只读） |
| 实付成本、实收金额 | 手动输入，可编辑 |
| 实际利润 | 自动计算：`实收金额 − 实付成本`，只读 |

**底部汇总（自动计算）：**
- 在表格最下方显示一行汇总：
- **「公司总利润」= 所有记录「实际利润」之和，单位：万元**
- 样式突出（加粗、醒目底色），实时更新

### 3.5 交互规范

- **确认弹窗：** 所有删除/迁移操作前，弹出 `confirm()` 确认框，明确告知将影响的记录数
- **空选择保护：** 未勾选任何记录时点击操作按钮，弹出提示"请先选择记录"
- **数据校验：** 迁移前检查必填字段是否为空，若为空则提示"请完善第 X 行 [字段名] 后再操作"
- **即时保存：** 任何增/删/改操作后自动写入 localStorage

每个视图页面有统一的页头（`.header`），包含返回主页链接和当前阶段标题。

## 5. 数据持久化

- 使用 `localStorage` 存储所有阶段的数据
- 存储键名：
  - `trade_bidding` — 投标阶段数据（JSON 数组）
  - `trade_won` — 中标阶段数据（JSON 数组）
  - `trade_completed` — 完成阶段数据（JSON 数组）
- 每个数组元素为对象，键名与字段对应（英文 key，如 `region`, `company`, `bank`, `amount` 等）
- 页面加载时从 localStorage 读取并渲染
- 数据变更后即时写入

## 6. 技术实现

- 纯 HTML + CSS + JavaScript，单文件

- 只读字段灰底显示

- 复选框列宽度固定 40px
