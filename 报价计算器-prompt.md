# 计算公式

## 共通第一步
```
贸易业务成本金额（元）= 业务金额 × 10000 × (贸易业务成本费率 ÷ 100)
```

## 分支A：成本付费方式 =「对私」
```
对私税筹费 = 贸易业务成本金额 ÷ (1 − 对私税筹费率 ÷ 100) − 贸易业务成本金额
报价金额   = (贸易业务成本金额 + 对私税筹费) ÷ (1 − 目标利润率 ÷ 100)
公司利润   = 报价金额 × (目标利润率 ÷ 100)
```

## 分支B：成本付费方式 =「对公」
```
对公开票费 = 贸易业务成本金额 ÷ (1 − 对公开票费率 ÷ 100) − 贸易业务成本金额
增值税     = (贸易业务成本金额 + 对公开票费) × (增值税率 ÷ 100)
报价金额   = (贸易业务成本金额 + 对公开票费 + 增值税) ÷ (1 − 目标利润率 ÷ 100)
公司利润   = 报价金额 × (目标利润率 ÷ 100)
```
